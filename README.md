# C Practical Exam Programs

This repository contains basic C language programs prepared for a practical examination.  
All programs are written in a simple and easy-to-understand manner.

## List of Programs

Q1 – Factorial.c  
Program to calculate the factorial of a given number.

Q2 – VowelCount.c  
Program to count the number of vowels in a given string.

Q3 – LeapYear.c  
Program to check whether a given year is a leap year or not.

Q4 – SwapusingPointer.c  
Program to swap two numbers using pointers.

Q5 – PatternPrinting.c  
Program to print patterns using loops.

## Language Used
C Programming Language

## How to Run
1. Compile the program:
   gcc filename.c  
2. Run the program:
   ./a.out

## Purpose
Academic practice and C practical examination reference.
